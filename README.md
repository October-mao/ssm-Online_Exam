# ssm-Online_Exam

#基于Spring+SpringMVC+Mybatis  SSM的在线考试系统

#管理员：增加老师以及学生，增删查改题目，生成试卷，导出试卷
#学生：进行考试，查询成绩，试卷回顾
#老师：生成试卷，增删查改试题，批量导入学生，增加学生，修改学生信息
